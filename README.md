# Отчёт: Two Sum

Глушков Матвей группа P3120

## Постановка задачи

Дан массив целых чисел `nums` и значение `target`. Требуется найти индексы двух чисел, сумма которых равна `target`. Один элемент нельзя использовать дважды.

## Алгоритм

Используется подход с хеш-таблицей (словарём) для однократного прохода по массиву.

### Идея

Для каждого числа `num` ищем **дополнение** `complement = target - num`. Если `complement` уже встречался ранее, пара найдена.

### Структура данных

- `seen: dict[int, int]` — словарь «значение → индекс» для уже просмотренных элементов.

### Шаги

1. Проходим по массиву с помощью `enumerate(nums)`.
2. Для текущего `num` вычисляем `complement = target - num`.
3. Если `complement` есть в `seen` — возвращаем `[seen[complement], i]`.
4. Иначе сохраняем `seen[num] = i` и продолжаем.
5. Если пара не найдена — возвращаем пустой список.

### Пример: nums = [2, 7, 11, 15], target = 9

| i | num | complement | seen до | действие |
|---|-----|------------|---------|----------|
| 0 | 2   | 7          | {}      | seen[2]=0 |
| 1 | 7   | 2          | {2:0}   | 2 в seen → return [0, 1] |

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(n) — хранение до n элементов в словаре.

## Тесты

| nums           | target | output  |
|----------------|--------|---------|
| [2, 7, 11, 15] | 9      | [0, 1]  |
| [3, 2, 4]      | 6      | [1, 2]  |
| [3]         | 6      | [0, 0]  |
